# SleepQualityTracker app using ROOM
The app was build during Android Kotlin Developer Nanodegree Certification
## SleepQualityTracker

The SleepQualityTracker app is a an app that helps you collect information about your sleep. 
* Start time
* End time
* Quality
* Time slept

This app demonstrates the following views and techniques:
* Room database
* DAO
* Coroutines

It also uses and builds on the following techniques from Android Architecture Components:
* Transformation map
* Data Binding in XML files
* ViewModel Factory
* Using Backing Properties to protect MutableLiveData
* Observable state LiveData variables to trigger navigation
